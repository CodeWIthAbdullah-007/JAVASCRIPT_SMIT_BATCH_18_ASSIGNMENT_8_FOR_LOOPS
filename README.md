# JAVASCRIPT_SMIT_BATCH_18_ASSIGNMENT_8_FOR_LOOPS

## Description
This repository contains 20 JavaScript exercises on FOR Loops for SMIT Batch 18.  
Each assignment uses `var`, `prompt`, and `alert` for beginner-friendly learning.  
Topics include: printing numbers, arrays, search, largest/smallest, multiples, loops, nested loops, decreasing values, even/odd checks, products, and star patterns.

## How to Run
1. Clone or download the repository to your computer.
2. Open the desired question folder, e.g., `Question_1`.
3. Open `index.html` in a web browser.
4. Interact with the prompt inputs to see alerts for each exercise.

## Notes
- All exercises are designed for beginners.  
- Uses only `var`, `prompt`, and `alert` for simplicity.  
- Each exercise demonstrates one or more FOR loop concepts, such as iteration, nested loops, array traversal, counting, and pattern generation.
